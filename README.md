# MinimapTweaks
A square minimap addon that contains a collection of tweaks such as size and scale adjustment, Blizz buttons reanchoring and more.

## Why this addon?
The goal of this addon is simple: change the map from circular to square and offer as much customizability as possible - nothing is "hardcoded". Do you want to hide the tracking icon but still see the calendar icon? No problem. Do you want to move the mail notification icon up, down, left, in the middle of the minimap? No problem! Hate having a border around the minimap? Or love having one? You get the idea.

## Features
Increase the scale of the minimap and all its elements via the minimap scale slider (same effect as edit mode scaling) or only change the size of the minimap itself without affecting its elements via the minimap size slider, or even combine both methods if you want!

Notable features, all of which can be toggled on or off independently:
- Minimap border with LibSharedMedia integration, size, offset and color settings
- Minimap zoom reset with configurable delay and zoom level to reset to
- Fully configure where default Blizz minimap icons appear, or toggle them off if desired: Addon Compartment, Battlefield, Calendar, Challenge Mode, Clock, Expansion Button, Instance Difficulty, Mail, Tracking, Voice Chat, Zone Text, Zoom in, Zoom out
- The Zone text can be fully configured
- Coordinates
- Custom Clock
- Custom Mouse bindings: Left/Middle/Right clicking on the Minimap, Custom Clock or Coordinate can be individually linked to different actions such as opening the tracking menu or the calendar

## Compatibility
This addon is not compatible with other addons that modify the minimap.
